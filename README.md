# Google Forms Lite Clone

A lightweight **Google Forms–style form builder** that allows users to create forms and collect responses.  
The project is organized as a **monorepo** containing both the **client** and **server** applications.

## Tech Stack

### Frontend
- TypeScript
- React
- Vite
- React Hook Form
- Redux Toolkit
- RTK Query
- React Router DOM
- Material UI

### Backend
- TypeScript
- GraphQL
- Apollo Server

## Project Overview

This application provides a minimal implementation of a form-building platform where users can:

- Create custom forms
- Add different types of questions
- Share forms with others
- Collect and submit responses

The project demonstrates a modern full-stack architecture using **React + GraphQL** with strong **TypeScript support**.


## Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
```
### 2. Navigate to the project directory
```bash
cd <project-folder>
```
### 3. Install dependencies
```bash
npm install
```
### 4. Run the project in development mode
```bash
npm run dev
```
This command starts both the client and server concurrently.
